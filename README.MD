# Rust Combustion Predictive Probe BLE Example

This is an example project demonstrating how to use the Combustion Inc. Predictive Probe BLE Specification with Rust.

The official combustion inc predictive probe ble specification can be found [here](https://combustioninc.com/predictive-probe-ble-specification), although that specification may not always be complete or up to date.

## Table of Contents

- [Installation](#installation)
- [Usage](#usage)
- [Support](#support)
- [Contributing](#contributing)

## Installation

Clone this repository:

```sh
git clone https://github.com/jwanga/rust-combustion-predictive-probe-ble-example.git
```

## Usage

To run the example, use the following command:

```sh
cargo run
```
